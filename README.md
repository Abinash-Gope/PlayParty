### 🛠️ Tech Stack & Architecture
* **Frontend**: `React 18` + `Vite` for sub-second hot module reloading.
* **Language**: `TypeScript` ensuring strict type-safety across client-side logic.
* **Styling**: `Tailwind CSS` & `shadcn/ui` for a premium, accessible component architecture.
* **State Management**: `React Query` & `React Context API` for seamless client caching.
* **Backend & Real-Time Sync**: `Supabase Realtime` (WebSockets) for synchronizing room state, active presence, and chat logs.
* **Media Integration**: `YouTube IFrame API` wrapped in custom synchronization hooks.

### ✨ Key Technical Features & Challenges Solved
1. **Real-time Playback Sync**: Implemented a millisecond-precision synchronization loop that reconciles playback state changes (play, pause, seek) across all peers in under 100ms.
2. **Dynamic Live Presence**: Integrated Supabase Presence trackers to display live peer activity, join/leave logs, and room participant counts.
3. **Synchronized Playback Queue**: Designed an optimistic queue updating system, allowing concurrent users to search, insert, delete, and re-order videos with instant UI updates while maintaining order consistency database-side.
